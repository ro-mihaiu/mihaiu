About Me

Learn more about my background, skills, and what drives me.

I’m Mihai, a curious developer from Iași, Romania, with a strong focus on backend systems, game servers, and web experiences. I enjoy turning ideas into practical setups, whether that means deploying a new game world, wiring up a database, making a script for a game or polishing the details of a personal site.

Most of my hands‑on experience comes from running and optimizing Minecraft servers, especially Paper based setups with plugins, scheduled tasks, and performance tuning on Linux hosts. I like treating these projects like real infrastructure: configuring restarts, managing resources, and experimenting with different server environments to keep things stable and fun for players.

On the development side, I work with HTML, CSS, and JavaScript on the frontend, and I’m comfortable using languages like Python, Java, C++, and Lua for scripting, tooling, or backend logic. For data, I use SQL with MariaDB and MySQL, and I run everything on Linux, which I use as my main environment for servers and experiments. I enjoy learning how the full stack fits together, from code to operating system.

I also spend time exploring cloud and VPS hosting, setting up services, panels, and databases, and generally tinkering with how things run under the hood. These projects helped me build confidence in server administration, networking basics, and troubleshooting real issues instead of just following tutorials.

Outside the terminal, I’m interested in community building and online experiences especially around gaming and tech. I like creating spaces where people can play, learn, and share what they’re working on, and I’m always looking for new projects that challenge me to grow as a developer and as a systems minded problem solver.
